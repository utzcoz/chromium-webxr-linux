WebXR test — VR session + browser overlay

This page tests both halves of the Linux patch in one place:

  1. Main WebXR: click Enter VR; the headset should show a solid dark-blue frame (the clear-color).
  2. Browser overlay: while still in VR, click any Permission or Capture button on the desktop. The Chromium prompt should appear as a small stereo quad near the bottom of each eye.

The on-page log records every event for correlation. If nothing appears in the headset, the log plus the --vmodule output tells you which layer failed.
